# 快速开始指南

## 安装依赖

在项目根目录下运行：

\`\`\`bash
npm install
\`\`\`

## 配置API密钥

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/) 注册账号
2. 进入"API Key"页面创建密钥
3. 创建 \`.env.local\` 文件（复制 \`.env.example\`）
4. 填入您的API密钥：

\`\`\`env
ZHIPU_API_KEY=your_actual_api_key_here
\`\`\`

或在应用启动后点击右上角设置按钮手动配置。

## 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

访问 [http://localhost:3000](http://localhost:3000)

## 使用说明

### 语音交互
- 点击底部麦克风按钮开始语音识别
- 对着麦克风说出您的问题
- 机器人会自动回复并朗读

### 文字交互
- 在左侧对话框中输入消息
- 点击"发送"按钮或按Enter键
- 查看机器人回复

### 3D交互
- 拖拽鼠标旋转机器人
- 滚动缩放视角
- 查看机器人不同角度

### 设置功能
- 点击右上角齿轮图标打开设置
- 配置API密钥
- 手动切换机器人表情
- 查看对话统计
- 清空对话历史

## 部署到Vercel

\`\`\`bash
npm run build
\`\`\`

然后：
1. 将代码推送到GitHub
2. 在Vercel导入项目
3. 设置环境变量 \`ZHIPU_API_KEY\`
4. 部署

## 常见问题

### 语音识别不工作？
- 使用Chrome/Edge浏览器
- 允许麦克风权限
- 检查浏览器设置

### 机器人不说话？
- 点击静音按钮取消静音
- 检查音量设置
- 确认浏览器支持Web Speech API

### API请求失败？
- 检查API密钥是否正确
- 确认网络连接正常
- 查看浏览器控制台错误信息
